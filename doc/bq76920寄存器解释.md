### 1. 系统状态与控制寄存器 (System Status & Control)

这些寄存器用于监控芯片状态、清除故障以及控制 MOS 管和 ADC 的开关。

| 寄存器名称  | 地址 | 位 (Bit) | 名称          | 读/写 | 复位值 | 功能详解                                                     |
| ----------- | ---- | -------- | ------------- | ----- | ------ | ------------------------------------------------------------ |
| SYS_STAT    | 0x00 | 7        | CC_READY      | RW    | 0      | 库仑计数器就绪。1=新的电流积分数据已准备好；0=无新数据。需写1清除。 |
|             |      | 6        | RSVD          | -     | 0      | 保留，勿用。                                                 |
|             |      | 5        | DEVICE_XREADY | RW    | 0      | 内部芯片故障。1=检测到内部逻辑错误（通常由瞬态干扰引起）。需写1清除。 |
|             |      | 4        | OVRD_ALERT    | RW    | 0      | ALERT 引脚外部强制。1=ALERT 引脚被外部电路拉高（如二级保护IC）。需写1清除。 |
|             |      | 3        | UV            | RW    | 0      | 欠压故障。1=检测到电池欠压。需写1清除。清除后需重新开启 DSG。 |
|             |      | 2        | OV            | RW    | 0      | 过压故障。1=检测到电池过压。需写1清除。清除后需重新开启 CHG。 |
|             |      | 1        | SCD           | RW    | 0      | 放电短路故障。1=检测到短路。需写1清除。清除后需重新开启 DSG。 |
|             |      | 0        | OCD           | RW    | 0      | 放电过流故障。1=检测到过流。需写1清除。清除后需重新开启 DSG。 |
| CELLBAL1    | 0x01 | 7-5      | RSVD          | R     | 0      | 保留。                                                       |
| (均衡控制1) |      | 4-0      | CB5-CB1       | RW    | 0      | 电芯均衡开关。1=开启对应电芯 (Cell 1~5) 的均衡；0=关闭。     |
| SYS_CTRL1   | 0x04 | 7        | LOAD_PRESENT  | R     | 0      | 负载存在检测。仅当 CHG_OFF 时有效。1=CHG 引脚被外部拉高，说明负载仍连接在 PACK- 上。 |
| (系统控制1) |      | 6-5      | RSVD          | R     | 0      | 保留。                                                       |
|             |      | 4        | ADC_EN        | RW    | 0      | ADC 使能。1=开启电压/温度测量（同时开启 OV/UV 保护）；0=关闭以省电。 |
|             |      | 3        | TEMP_SEL      | RW    | 0      | 温度源选择。0=读取内部芯片温度；1=读取外部热敏电阻 (TSx) 温度。 |
|             |      | 2        | RSVD          | RW    | 0      | 保留，必须写 0。                                             |
|             |      | 1-0      | SHUT_A/B      | RW    | 0      | 关机序列控制。进入 SHIP 模式需按特定顺序写入： 1. 写 `0x02` (A=0, B=1) 2. 写 `0x04` (A=1, B=0)。 |
| SYS_CTRL2   | 0x05 | 7        | DELAY_DIS     | RW    | 0      | 禁用保护延时。1= bypass OV/UV/OCD/SCD 的延时逻辑（用于产线快速测试），故障立即触发。正常使用时应为 0。 |
| (系统控制2) |      | 6        | CC_EN         | RW    | 0      | 库仑计数器常开。1=持续进行电流积分；0=关闭常开模式。         |
|             |      | 5        | CC_ONESHOT    | RW    | 0      | 单次电流测量。1=触发一次 250ms 的电流测量，完成后自动清零。仅在 CC_EN=0 时有效。 |
|             |      | 4-2      | RSVD          | RW    | 0      | 保留，必须写 0。                                             |
|             |      | 1        | DSG_ON        | RW    | 0      | 放电 MOS 管控制。1=开启放电通路；0=关闭。故障发生后不会自动恢复，需软件写入 1。 |
|             |      | 0        | CHG_ON        | RW    | 0      | 充电 MOS 管控制。1=开启充电通路；0=关闭。故障发生后不会自动恢复，需软件写入 1。 |



### 2. 保护配置寄存器 (Protection Configuration)

用于设置过流、短路的阈值和延时，以及过压、欠压的阈值和延时。

| 寄存器名称     | 地址 | 位 (Bit) | 名称       | 功能详解                                                     |
| -------------- | ---- | -------- | ---------- | ------------------------------------------------------------ |
| PROTECT1       | 0x06 | 7        | RSNS       | 量程倍增。0=低量程 (OCD/SCD 阈值较低)；1=高量程 (阈值 x2)。 配合下方阈值位使用。 |
| (短路保护1)    |      | 6        | RSVD       | 保留。                                                       |
|                |      | 5        | RSVD       | 保留。                                                       |
|                |      | 4-3      | SCD_DELAY  | 短路延时设置: `00`=70µs, `01`=100µs, `10`=200µs, `11`=400µs。 |
|                |      | 2-0      | SCD_THRESH | 短路阈值设置 (单位 mV，取决于 RSNS 位): 若 RSNS=0: `000`=22mV ... `111`=100mV 若 RSNS=1: `000`=44mV ... `111`=200mV |
| PROTECT2       | 0x07 | 7        | RSVD       | 保留。                                                       |
| (过流保护2)    |      | 6-4      | OCD_DELAY  | 过流延时设置: `000`=8ms ... `111`=1280ms (具体查表，呈倍数增长)。 |
|                |      | 3-0      | OCD_THRESH | 过流阈值设置 (单位 mV，取决于 RSNS 位): 若 RSNS=0: `0000`=8mV ... `1111`=50mV 若 RSNS=1: `0000`=17mV ... `1111`=100mV |
| PROTECT3       | 0x08 | 7-6      | UV_DELAY   | 欠压延时设置: `00`=1s, `01`=4s, `10`=8s, `11`=16s。          |
| (压延保护延时) |      | 5-4      | OV_DELAY   | 过压延时设置: `00`=1s, `01`=2s, `10`=4s, `11`=8s。           |
|                |      | 3-0      | RSVD       | 保留，保持默认。                                             |
| OV_TRIP        | 0x09 | 7-0      | OV_THRESH  | 过压阈值中间8位。 实际阈值为 `10` + [这8位] + `1000` (共14位 ADC 值)。 需结合 ADCGAIN/OFFSET 计算具体电压值 (约 3.15V~4.7V)。默认 0xAC。 |
| UV_TRIP        | 0x0A | 7-0      | UV_THRESH  | 欠压阈值中间8位。 实际阈值为 `01` + [这8位] + `0000` (共14位 ADC 值)。 需结合 ADCGAIN/OFFSET 计算具体电压值 (约 1.58V~3.1V)。默认 0x97。 |
| CC_CFG         | 0x0B | 7-6      | RSVD       | 保留。                                                       |
| (CC 配置)      |      | 5-0      | CC_CFG     | 必须写入 0x19 以获得最佳库仑计数器性能。                     |



### 3. 数据读取寄存器 (Data Readout Registers)

这些寄存器只读，存储最新的测量数据。读取时应连续读取 HI 和 LO 字节。

| 寄存器组    | 地址范围    | 说明                                                         |
| ----------- | ----------- | ------------------------------------------------------------ |
| VC1 ~ VC5   | 0x0C - 0x15 | 电芯电压 1-5。每节电芯占用 2 字节 (HI/LO)。格式为 14-bit ADC 值。 例如：VC1_HI (0x0C), VC1_LO (0x0D)。 |
| VC6 ~ VC10  | 0x16 - 0x1F | 电芯电压 6-10 (仅 BQ76930/40 有效)。                         |
| VC11 ~ VC15 | 0x20 - 0x29 | 电芯电压 11-15 (仅 BQ76940 有效)。                           |
| BAT         | 0x2A - 0x2B | 电池组总电压。16-bit 值，LSB ≈ 1.532mV。                     |
| TS1         | 0x2C - 0x2D | 温度 1。可配置为外部热敏电阻或内部芯片温度。14-bit ADC 值。  |
| TS2         | 0x2E - 0x2F | 温度 2 (仅 BQ76930/40 有效)。                                |
| TS3         | 0x30 - 0x31 | 温度 3 (仅 BQ76940 有效)。                                   |
| CC          | 0x32 - 0x33 | 库仑计数器 (电流积分)。16-bit 有符号数 (2's complement)。 电压值 = 数值 × 8.44µV。电流 = 电压 / Rsense。 |



### 4. 校准系数寄存器 (Calibration Coefficients)

这些寄存器只读，存储出厂校准的 Gain 和 Offset，用于将 ADC 原始数据转换为精确电压。

| 寄存器名称 | 地址 | 位 (Bit) | 说明                                                         |
| ---------- | ---- | -------- | ------------------------------------------------------------ |
| ADCGAIN1   | 0x50 | 4-3      | ADC Gain 高2位。                                             |
| ADCGAIN2   | 0x59 | 7-5      | ADC Gain 低3位。 组合成 5-bit 值，计算公式： $ Gain (\mu V/LSB) = 365 + Value $ 。范围 365~396。 |
| ADCOFFSET  | 0x51 | 7-0      | ADC Offset。8-bit 有符号数 (mV)。范围 -128mV ~ +127mV。      |
